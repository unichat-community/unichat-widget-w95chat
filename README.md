A Windows 95 themed chat widget for **UniChat**.

Supports the following events:
- Message Removed
- User Removed
- Chat Cleared
- Message
- Donation (YouTube super chat/sticker, Twitch bits)
- Sponsorship (YouTube Members, Twitch Subscription)
- SponsorGift (YouTube Member Gift, Twitch Subscription Gift)
- Raid (YouTube and Twitch)
- Redemption (Twitch Channel Redemption)

> [!NOTE]
> Just like any other **UniChat** widget, this widget also works with sources originating from plugins (like kick through `plugin-kick`),
> but it was primarily made to work with native sources (YouTube and Twitch), so there may be limitations and/or unexpected behaviors when using plugin sources.
